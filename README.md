## 📖 项目简介 (Introduction)

欢迎来到我的 **CS61A (2024 Fall)** 学习仓库！
这是 UC Berkeley 计算机科学入门课程的代码库，涵盖了函数式编程、面向对象编程、解释器构造以及声明式编程等核心计算机科学概念。

在这个仓库中，我记录了所有的作业 (Homework)、实验 (Lab) 以及课程项目 (Projects) 的代码实现。

## 🏗️ 核心项目 (Projects)

CS61A 的精髓在于它的四个大型编程项目，本项目已包含：

| 项目名称 | 简介 | 涉及技术 | 目录 |
| :--- | :--- | :--- | :--- |
| 🎲 **Hog** | 基于骰子的策略游戏。通过实现游戏逻辑和高阶函数来制定最佳获胜策略。 | `Python`, `Higher-Order Functions` | `./Project/hog/` |
| 🐱 **Cats** | 一个打字速度测试与自动纠错程序。涉及复杂的递归逻辑和搜索算法。 | `Python`, `Recursion`, `Search` | `./Project/cats/` |
| 🐜 **Ants** | 类似于《植物大战僵尸》的塔防游戏。深入理解面向对象编程 (OOP) 和继承。 | `Python`, `OOP`, `Inheritance` | `./Project/ants/` |
| 🔮 **Scheme** | 用 Python 编写的一个 Scheme 语言解释器。探索编程语言的底层实现、求值环境与尾递归优化。 | `Python`, `Scheme`, `Interpreter` | `./Project/scheme/` |
